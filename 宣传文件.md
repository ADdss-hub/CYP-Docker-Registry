# CYP-Docker Registry：零信任容器镜像仓库，让安全与高效兼得
专为个人开发者与小型团队打造的企业级容器镜像管理解决方案——CYP-Docker Registry v1.0.1 重磅登场！基于零信任架构重构，融合供应链安全、智能加速、全场景适配能力，无需复杂配置，开箱即享安全可控的镜像管理体验。

## 核心亮点：安全为基，体验为王
✅ **零信任安全防护，杜绝无授权访问**  
强制登录认证贯穿全链路，API/界面/资源均需身份校验，3次登录失败、5次令牌错误或10次未授权API访问即触发自动锁定。内置入侵检测引擎（IDS），精准识别伪造JWT、IP异常变更、绕过登录等风险行为，搭配硬件资源限制、网络封锁、服务暂停三重锁定机制，从根源抵御攻击。区块链哈希审计日志不可篡改，1年数据留存，合规追溯无压力。

✅ **极致传输性能，速度提升50-80%**  
P2P去中心化分发+分层压缩（zstd/gzip自适应）+智能缓存三重加速，NAT自动穿透打破网络壁垒，即使低功耗设备也能高效传输。支持本地/云/混合存储智能切换，无需手动干预存储策略。

✅ **全链路供应链安全，镜像可信可溯**  
自动生成SBOM软件物料清单，集成Trivy漏洞扫描，实时预警CRITICAL/HIGH级风险；通过Sigstore/cosign强制镜像签名，杜绝恶意篡改，每一次推送、拉取都有安全背书。

✅ **全平台适配+零配置体验，部署无门槛**  
从X86到ARM，从树莓派、NAS到K8s、云服务器，一键启动无需适配。内置环境感知引擎，自动识别Docker/阿里云/群晖/物理机等运行场景，动态优化配置，新手也能快速上手。

✅ **轻量化架构+自动化运维，减负增效**  
基于Go+Vue3构建，轻量无冗余， SQLite零配置数据库降低部署成本；支持自动更新、镜像清理、签名扫描全流程自动化，减少80%手动运维工作。团队协作更高效——组织命名空间+RBAC权限控制+密码保护分享链接，灵活管理访问边界。

## 适用场景
- 个人开发者：安全存储个人项目镜像，免担心泄露风险  
- 小型团队：低成本实现团队镜像共享、版本管控与权限隔离  
- 边缘部署：适配树莓派等低功耗设备，满足边缘计算场景需求  
- 合规需求：符合GDPR/HIPAA/SOC2合规基线，审计日志可追溯  

## 快速上手，3步启动
1. 克隆项目：`git clone https://github.com/CYP/cyp-docker-registry.git`
2. 启动服务：`cd cyp-docker-registry && docker-compose up -d`
3. 访问Web界面（http://localhost:8080），完成初始登录即可使用  

## 技术背书，企业级保障
- 跨平台编译：Go 1.21+支持全架构部署
